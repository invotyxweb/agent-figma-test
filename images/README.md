# Images Directory

This directory contains all the images used in the landing page.

## Image Specifications

- **Hero Image**: Main banner image for the hero section
- **Feature Icons**: Icons representing different features
- **About Images**: Images for the about section
- **Logo**: Company logo in various formats

## Recommended Formats

- **PNG**: For images with transparency
- **JPEG**: For photographs and complex images
- **SVG**: For icons and simple graphics (scalable)

## Image Guidelines

- Optimize all images for web (compress file sizes)
- Use responsive images with appropriate sizes
- Include alt text for accessibility
- Consider using WebP format for better compression
